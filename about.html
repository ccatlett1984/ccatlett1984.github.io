About SeAT

Since the very early days of me becoming a CEO in Eve Online, the need for a API Management Tool became important. Basical tools for keeping record of API Keys in a easily, recallable way proved to be lacking. The options that were available, were old, outdated, or not exactly what I wanted.

Once the API inventory problem was solved (generally via Google Docs), the next hurdle of knowing which keys are valid, which keys belong to who etc. come to light. All of these were challenged I faced, and inspired me to write tools to help me solve them.

SeAT is actually the product of almost 3 complete tool rewrites. However, it is by far the most superior one and has exploded in features thanks to a healthy community of open source developers giving their time.
